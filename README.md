# strata_hub

Strata Hub – Identity & Access Management
Role: Acts as the central identity provider and administrative interface.

Functions:

- Manages user authentication via external identity providers including Google, ClassLink, and Clever.

- Organizes users into organizations, groups, and roles (e.g., admin, leader, player).

- Controls which games are assigned to which groups, enabling access permissions for players.

- Dynamically renders content for authenticated and non-authenticated users using HTMx and TailwindCSS.
